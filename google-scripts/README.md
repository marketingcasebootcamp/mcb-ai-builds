# Google Scripts

Google Apps Script tools for Sheets and other Google products.

---

## What lives here

Each script is a `.gs` file you paste into the Apps Script editor.
Some are paired with a Google Sheet template (link in the module README).

---

## How to use a script

1. Open the module folder and read `setup-instructions.md`
2. Open Google Apps Script (script.google.com or via Tools → Apps Script in Sheets)
3. Paste the contents of `Code.gs`
4. Follow the setup instructions
