# Contributing to mcb-ai-builds

Read the full [Contribution Guide →](./docs/contribution-guide.md) before submitting.

Quick start:
1. Open a module proposal using the [issue template](https://github.com/marketingcasebootcamp/mcb-ai-builds/issues/new/choose)
2. Wait for a green label — idea is approved to build
3. Fork the repo and create a branch: `add/[module-slug]`
4. Build the module following the [Module Spec →](./docs/module-spec.md)
5. Open a pull request using the PR template
