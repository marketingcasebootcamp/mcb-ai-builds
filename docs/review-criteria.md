# Review criteria

---

## Universal checks (all types)

- [ ] Solves a real, repeatable marketing task
- [ ] README has all required sections
- [ ] metadata.yaml is complete and valid
- [ ] examples/ has at least one real input/output pair
- [ ] No real API keys in any committed file

---

## Automatic rejection

- Real API key in any committed file
- No tested examples
- Duplicate module without meaningful improvement
- Incomplete metadata
