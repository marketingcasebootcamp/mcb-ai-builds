# Apps

Interactive tools for marketing work — Streamlit apps and browser-based web tools.

---

## Two types

### [Streamlit →](./streamlit/)

Python apps that run locally or deploy to Streamlit Cloud.
Good for data-heavy tools: calculators, validators, AI-powered processors.

### [Web →](./web/)

Browser-based tools built with HTML, React, or Next.js.
Most open directly in a browser with no setup.

---

## Apps that use AI

Some apps require an API key.
Read the [API Key Setup Guide →](../docs/api-setup.md) before running any AI-powered app.
**Never put a real API key in any code file.**
