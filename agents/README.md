# Agents

This folder will contain MCP (Model Context Protocol) servers
and autonomous Claude agents for marketing workflows.

---

## What this will be

MCP servers expose marketing tools directly to Claude —
so instead of opening a separate app, you just talk to Claude
and it calls the right tool mid-conversation.

Examples we're planning:
- mcb-campaign-tools MCP server — wraps several marketing calculators
- Campaign audit agent
- Keyword research agent

---

## Status

In progress. We're building toward this once the apps layer is mature.
